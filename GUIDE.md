# Designt.io 使用指南

> 這份指南會帶你從零開始學會使用 Designt.io，快速製作出專業水準的投影片。

---

## 哪種模式適合你？

Designt.io 提供兩種主要的工作模式，適合不同階段的使用者：

| | 專案編輯器 | AI 對話模式 |
|---|---|---|
| **一句話描述** | 把現有素材變成精美投影片 | 從零開始，讓 AI 幫你規劃與設計 |
| **適合誰** | 已經有 PPT 草稿、截圖或文字稿的人 | 只有一個主題或想法，還沒規劃具體內容的人 |
| **輸入方式** | 上傳圖片 / Word / TXT | 用對話描述需求 |
| **核心流程** | 上傳 → 選風格 → 一鍵重繪 | 對話 → AI 規劃 → 編輯 → 生成圖片 |
| **局部修改** | ✅ 支援（塗抹遮罩） | ❌ 需轉存為專案後使用 |

### 選擇建議

- **「我已經做好 PPT 初稿，想讓它更好看」** → 用 **專案編輯器**。把每頁截圖上傳，選一個模板風格，一鍵重繪全部頁面。

- **「我有文字大綱，想直接生成投影片」** → 用 **專案編輯器**。用「上傳 Word / TXT」匯入文字稿，搭配風格參考圖生成。

- **「我只有一個主題，不知道每頁要放什麼」** → 用 **AI 對話模式**。告訴 AI 你的簡報主題和目的，讓它幫你規劃每頁的標題和內容，再一鍵生成圖片。

- **「我有參考資料，想讓 AI 幫我整理成簡報」** → 用 **AI 對話模式**。上傳你的 Word、PDF、Excel 等資料作為附件，AI 會根據內容規劃投影片。

- **「我想先在對話中構思，之後再精修」** → 先用 **AI 對話模式** 完成規劃和初步生成，再點 **存為專案** 轉到專案編輯器做局部修改和微調。

> 💡 **兩種模式可以串接**：在 AI 對話中完成規劃後，隨時可以「存為專案」轉到專案編輯器，享受更完整的編輯功能（局部修改、撤銷重做等）。

## 1. 登入與首頁

1. 開啟網站後會看到歡迎頁面，點擊 **Get Started** 或 **Sign In**。
2. 選擇 **Google 帳號登入** 或 **Email 登入**。
3. 登入後自動跳轉到 **專案列表** 首頁。

---

## 2. 建立與管理專案

### 建立新專案

1. 在首頁點擊 **＋ 新增專案** 卡片。
2. 輸入專案名稱。
3. 選擇喜歡的顏色和圖示（可選）。
4. 點 **建立** 完成。

### 管理專案

- **進入專案** — 點擊專案卡片。
- **編輯名稱 / 顏色 / 圖示** — 點專案卡片右上角的 ✏️ 編輯按鈕。
- **刪除專案** — 點專案卡片右上角的 🗑️ 按鈕，確認後刪除（無法復原）。

---

## 3. 專案編輯器 — 上傳素材

進入專案後會看到編輯器，左側是投影片列表，右側是編輯區域。

### 方法一：上傳圖片（PPT 截圖）

1. 點左下角 **＋** 按鈕 → 選擇 **上傳圖片**。
2. 選取一或多張投影片截圖（支援 JPG / PNG）。
3. 每張圖片會自動建立一頁投影片。

### 方法二：上傳文字檔（Word / TXT）

1. 點左下角 **＋** 按鈕 → 選擇 **上傳 Word/TXT**。
2. 文字檔內容需用「第一頁」「第二頁」等標記分頁，格式範例：

```
第一頁
標題：損益表的真相
重點：現金流比獲利更重要

第二頁
標題：資產負債表解析
重點：資產 = 負債 + 股東權益
```

3. 系統會自動解析並建立對應的文字投影片。

### 方法三：手動新增空白頁

1. 點左下角 **＋** 按鈕 → 選擇 **新增空白頁**。
2. 在右側的文字框中輸入投影片內容。

### 調整順序

- 在左側投影片列表中 **拖曳** 投影片縮圖即可調整順序。

---

## 4. 專案編輯器 — 設定風格

生成前需要設定風格，讓 AI 知道你想要什麼風格的投影片。

### 選擇風格參考圖（重要）

1. 點右側面板的 **模板庫** 按鈕。
2. 瀏覽模板，找到喜歡的風格後點擊 **套用**。
3. 也可以點 ⭐ 收藏常用模板。
4. **歷史紀錄** 分頁會記住你之前用過的模板。

> 💡 風格參考圖會決定生成結果的整體視覺風格，是影響品質最關鍵的一步。

### 進階設定（可選）

點頂部工具列的齒輪圖示或在面板中展開進階設定：

| 設定項 | 說明 | 範例 |
|--------|------|------|
| 字體 | 指定字體系列 | Noto Sans TC |
| 主色 | 文字主體顏色 | 黑色、#333333 |
| 強調色 | 重點標示方式 | 藍色底線、紅色粗體 |
| 背景色 | 投影片背景 | 白色、淺灰 |
| 特殊標記 | 浮水印等 | 校徽浮水印、公司 LOGO |
| 額外提示詞 | 補充指示 | 不要太花俏，背景簡潔 |
| 比例 | 投影片尺寸 | 16:9（預設）、1:1、4:3 |
| 解析度 | 輸出品質 | 1K（快速）、2K、4K（高畫質） |

## 5. 專案編輯器 — 生成投影片

### 選取要生成的投影片

- 點擊左側投影片縮圖選取，被選中的會有藍色外框。
- 可按住 Shift 或 Ctrl 多選。
- 頂部有 **全選所有頁面** 按鈕可一鍵全選。

### 開始生成

1. 確認已設定風格參考圖。
2. 點擊 **1-Click Modify** 按鈕。
3. AI 開始逐張生成，可在進度條看到目前進度（例如 3/10）。
4. 生成完成的投影片會在左側顯示生成結果。

### 生成中的操作

- **取消生成** — 點 ✕ 停止按鈕。
- 遇到 429 錯誤（API 使用量過高）會自動重試最多 3 次，每次間隔 5 秒。
- 如果仍然失敗，會跳出自動重試設定面板，可以：
  - 設定重試間隔（秒）
  - 設定停止條件（成功後停止 / 達到次數上限 / 到指定時間）

### 撤銷與重做

- 生成後如果不滿意，點投影片右上方的 **↩ 撤銷** 回到上一版。
- 點 **↪ 重做** 回到較新的版本。
- 點 **回到原稿** 回到最初上傳的圖片。

## 6. 專案編輯器 — 局部修改

如果只想修改投影片的某個區域（例如某段文字寫錯、某張圖想換）：

1. 選取要修改的投影片。
2. 在生成的圖片上 **用滑鼠塗抹** 想要修改的區域（會出現半透明的遮罩）。
3. 在下方文字框輸入修改指令（例如「把標題改成 XXX」）。
4. 點 **局部修改** 按鈕。
5. AI 只會重新生成你塗抹的區域，其餘保持不變。

## 7. 專案編輯器 — 匯出與下載

### 下載圖片

1. 點頂部工具列的 **下載圖片** 按鈕。
2. 選擇 **全部頁面** 或 **已選取頁面**。
3. 瀏覽器支援的話會讓你選擇存放資料夾；否則會自動下載 ZIP 壓縮檔。

### 匯出 PPTX

1. 點頂部工具列的 **匯出 PPTX** 按鈕。
2. 選擇匯出範圍（全部 / 已選取）。
3. 自動下載 `.pptx` 檔案，可用 PowerPoint 或 Google Slides 開啟。

### 預覽模式

- 點頂部 **預覽** 按鈕，全螢幕瀏覽所有已生成的投影片。
- 點 **退出預覽** 回到編輯模式。

### 備份

- 生成的圖片會 **自動備份** 到雲端（生成後約 2 秒自動開始）。
- 頂部會顯示 **備份 (N)** 按鈕，也可手動點擊觸發。
- 備份完成後顯示 ✓ 已備份。

## 8. AI 對話模式

從側邊欄點擊 **AI 對話** 進入。適合還沒有素材、想從頭開始規劃簡報的情境。

### 基本對話

1. 在底部輸入框輸入你的需求，例如：「幫我做一份 10 頁的公司季報簡報」。
2. 可以用 📎 按鈕上傳附件（圖片、Word、PDF、Excel 等）。
3. AI 會根據你的需求回覆建議。

### 投影片規劃

1. 在對話中描述簡報需求後，點 **📋 規劃投影片** 按鈕。
2. 設定頁數後，AI 會自動產生投影片規劃（每頁的標題 + 內容大綱）。
3. 規劃結果顯示在下方的 **投影片規劃面板** 中。

### 編輯規劃內容

- 點左側列表切換頁面，在右側編輯標題和內容。
- **新增頁面** — 點 ＋ 按鈕，設定要新增的頁數。
- **刪除頁面** — 點該頁右上角的 🗑️ 按鈕。

### 文字工具列

在每頁內容下方有快捷工具按鈕：

| 按鈕 | 功能 |
|------|------|
| 📝 擴寫 | AI 自動擴充內容 |
| ✂️ 精簡 | AI 精簡濃縮內容 |
| 🎭 換語氣 | 正式 / 輕鬆 / 學術 / 幽默 |
| 🔍 搜尋擴充 | 使用 Google 搜尋補充最新資料 |

### 生成投影片圖片

1. 點 **模板庫** 選取風格參考（與專案編輯器相同）。
2. 點 ⚙️ **設定** 調整字體、顏色、比例等。
3. 點 ▶️ **生成** 按鈕，AI 會根據規劃逐頁生成投影片圖片。
4. 生成的圖片顯示在右側 **圖片面板**。

### 匯出

- **匯出 PPTX** — 點右側面板的匯出按鈕，將文字稿 + 圖片打包成 PowerPoint。
- **存為專案** — 點投影片規劃面板的 📁 **存為專案** 按鈕，輸入名稱後：
  - 自動建立新專案
  - 文字稿成為投影片內容
  - 已生成的圖片直接帶入
  - 跳轉到專案編輯器，可繼續編輯

### 對話管理

- **新對話** — 點左側歷史面板的 ＋ 按鈕。
- **搜尋對話** — 頂部搜尋框。
- **重新命名** — 將滑鼠移到對話項目上，點 ✏️。
- **刪除對話** — 將滑鼠移到對話項目上，點 🗑️。

## 9. 設定頁

從側邊欄點擊 **設定** 進入。

### 外觀

- 點選 **淺色模式** 或 **深色模式** 即時切換。

### API 設定

- **Gemini API Key**（選填）— 留空會使用平台預設的 API，不需額外付費。如果填入自己的 Key，生成費用會由 Google 向你計費。
- **模型選擇**：
  - **Gemini 3.1 Flash Image** — 速度快，適合一般使用（預設）。
  - **Gemini 3 Pro Image** — 品質較高，速度稍慢。
- 設定完後點 **儲存設定**。

### Google Drive 備份（選填）

- 在設定頁填入 Google Drive Script URL 後，生成的圖片會額外備份一份到你的 Google Drive。

## 10. 常見問題

### Q：生成時出現「429 錯誤」怎麼辦？

這表示 API 使用量過高。系統會自動重試，你也可以：
- 等待 5～10 分鐘後再試。
- 使用自動重試功能，設定間隔時間讓系統自己重試。

### Q：生成結果不滿意怎麼辦？

- **重新生成** — 選取該頁後再次點 1-Click Modify。
- **局部修改** — 在圖上塗抹不滿意的區域，輸入修改指令。
- **換風格** — 更換模板庫中的風格參考圖再重新生成。
- **調整設定** — 修改額外提示詞，加上更明確的指示。

### Q：圖片生成後消失了？

生成的圖片會自動備份到雲端。如果看到「備份 (N)」按鈕，表示還有未備份的圖片，請等待或手動點擊備份。備份完成後，重新整理頁面圖片仍會存在。

### Q：可以修改已生成投影片的文字內容嗎？

可以。在文字投影片中直接編輯文字框內容，然後重新生成即可。也可以使用 AI 潤色功能自動優化文字。

### Q：如何從 AI 對話模式切換到專案模式？

在 AI 對話的投影片規劃面板中，點 📁 **存為專案** 按鈕，即可將所有規劃內容和已生成的圖片匯入新專案。
